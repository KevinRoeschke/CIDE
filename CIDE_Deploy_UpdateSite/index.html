CIDE Update Site

How to use this update site

Open Eclipse 3.4.
From the Help menu select Software Updates...
Go to Available Software tab
Select Add Site button
Set location to http://wwwiti.cs.uni-magdeburg.de/iti_db/research/cide/update/ and click ok
Select either CIDE (old) or CIDE (Core). If you select the latter, you can additionally install optional plugins.
